Hid.im Firefox Extension
by Michael Nutt <michael@nuttnet.net>

This Firefox extension decodes hid.im images into torrents.  To use it first
install it in Firefox and then right-click on a hid.im image and there should
be a "Save Torrent As..." option in the context menu.
